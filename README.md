Developing
==========

Use librarian-chef gem:

`gem install librarian-chef`

`librarian-chef install`

`vagrant box add  chef/centos-6.5 https://developer.nrel.gov/downloads/vagrant-boxes/CentOS-6.5-x86_64-v20140504.box`

`vagrant up`

`vagrant provisions`
